WHATSAPP BOT PRIVADO

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> npm install
> git clone https://github.com/vinimods2/botp1
> ls
> cd botp1
> npm i -g pm2
> pm2 start index.js
> pm2 monit
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```
